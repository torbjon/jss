# Rails JavaScriptS collection

## Install

    gem "jss"

## Require

Add the following to your `app/assets/javascripts/application.js`:

[json2](https://github.com/douglascrockford/JSON-js)

    //= require json
    //= require json2

[html5shiv](https://github.com/aFarkas/html5shiv)

    //= require html5shiv
